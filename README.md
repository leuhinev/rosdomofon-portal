# РосДомофон Портал

Web-портал для управления номерами автомобилей и ключами домофона, интегрированный с API РосДомофона.

## Функционал

- Авторизация по номеру телефона с подтверждением через push-уведомление
- Управление номерами автомобилей (добавление, редактирование, удаление)
- Управление ключами домофона Miware Classic
- Автоматическая синхронизация с данными РосДомофона
- JWT-авторизация с 60-дневными сессиями
- Встроенный фронтенд (один бинарный файл)

## Требования

- Go 1.22+
- MySQL 8+
- Memcached

## Установка

1. Скопируйте `config.yaml.example` в `config.yaml` и заполните настройки
2. Запустите MySQL и Memcached
3. Выполните миграции: `make migrate-up`
4. Запустите приложение: `make run`
